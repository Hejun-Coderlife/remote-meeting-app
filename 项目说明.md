- Socket.io Client
- WebRTC API

**后端技术栈：**
- Node.js + Express
- Socket.io (WebSocket服务器)
- 房间和用户管理

### 🚀 如何启动

1. **快速启动：**
   ```bash
   cd remote-meeting-app
   ./start.sh
   ```

2. **手动启动：**
   ```bash
   cd remote-meeting-app
   npm run install-all  # 安装所有依赖
   npm run dev          # 启动开发服务器
   ```

3. **访问应用：**
   - 前端：http://localhost:3000
   - 后端：http://localhost:3001

### 💡 使用说明

1. **创建房间：** 输入姓名和房间名称，点击"创建房间"
2. **加入房间：** 输入姓名和房间ID，点击"加入房间"
3. **会议功能：**
   - 🎤 点击麦克风图标控制静音
   - 📹 点击摄像头图标控制视频
   - 🖥️ 点击屏幕共享图标分享桌面
   - 💬 在右侧聊天框发送消息
   - 📞 点击红色电话图标离开会议
